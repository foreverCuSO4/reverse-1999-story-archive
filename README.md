# Reverse: 1999 Story Archive / 重返未来：1999 剧情整理

This repository collects a cleaned bilingual story archive for **Reverse: 1999**, covering versions **1.0 to 3.6**.

本仓库收录了 **《重返未来：1999》1.0 到 3.6** 的整理版剧情文本，并提供多种便于阅读和下载的格式。

## Contents / 内容

- `markdown/`: bilingual Markdown files for each major chapter or event version
- `txt/`: plain text exports generated from the Markdown files
- `pdf/`: printable PDF exports generated from the same cleaned source files

- `markdown/`：每个大章或活动版本对应一份中英双语 Markdown
- `txt/`：由 Markdown 导出的纯文本版本
- `pdf/`：由同一套整理文本导出的 PDF 版本，便于阅读与下载
